Code repository for Shan*, Minni* and Duncker, "Separating the what and how of compositional computation to enable reuse and continual learning", NeurIPS 2025. https://arxiv.org/abs/2510.20709.

* `task_model.py`: learning and inference of the task model ("what" system)
* `tast_model_demo.ipynb`: notebook demo of learning and inference of the task model
* `train.py`: training the low-rank RNN ("how" system) together with the task model
* `example_trial.ipynb`: plot the inputs and outputs of an example trial
